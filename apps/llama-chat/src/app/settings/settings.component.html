🚀 Starting the LLM Server

Run one of the following commands from the repo root to download and start llama.cpp with the chosen model (Apple M5 — Metal GPU offload enabled automatically):

Gemma 4 Obliterated (Q8_0) — recommended
npx nx start llm
Gemma 4 Uncensored (Q6_K_P)
npx nx start llm --configuration=uncensored

The model GGUF is cached in ~/.cache/llama.cpp/ after the first download. Set HF_TOKEN in your environment if the repo requires authentication.
